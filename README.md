# Gra Zgadnij Liczbę
Prosta i zabawna, klasyczna gra w odgadywanie liczby, idealna do wspólnej zabawy z dziećmi.

## Jak grać
Gra wspiera rozgrywkę dla wielu graczy, którzy kolejno odgadują tajną liczbę. Po każdej próbie aktywny gracz otrzymuje wskazówkę — czy jego liczba jest zbyt wysoka, zbyt niska, czy poprawna.

Po zakończeniu wybranej liczby rozgrywek wyświetlana jest tabela wyników pokazująca średnią liczbę prób każdego z graczy.
